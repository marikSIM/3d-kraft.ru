Как пользоваться. Первое что нужно сделать это загрузить свой фай-STL.
Затем выставляете время печати который можно посмотреть в любом слайсере.
Далее вводите сумму моделирования - если моделировали сами. 
Выбираете материал из которого будет производится печать и по необходимости регулируете плотность материала.
Всё, получаете сумму с учётом налогооблажения и мото часами!.
